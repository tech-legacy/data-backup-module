分布式数据加密备份

## Features

- [x] Automatic File Backup 自动文件备份
- [ ] Source Folder 待备份文件夹
- [ ] Destination Folder 备份目标文件夹 (可选压缩、加密)
- [ ] Temporary Decryption Folder 临时解密文件夹 (备份数据解密时可选仅保留24小时)
